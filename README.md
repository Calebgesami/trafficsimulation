

# 🚦 Traffic Simulation System

A Python-based traffic simulation designed to model and analyze vehicle movement at road intersections and evaluate traffic signal strategies such as **fixed-time signals** and **adaptive traffic control**.

The project aims to help understand how different traffic light algorithms affect **vehicle flow, congestion, and travel time**.

---

# 📌 Features

* 🚗 Vehicle generation and movement simulation
* 🚦 Traffic light control system
* 📊 Performance metrics (average travel time, vehicle throughput)
* 🔄 Comparison between **fixed traffic signals** and **adaptive signals**
* 📈 Traffic performance analysis

---

# 🧠 Project Goal

Traffic congestion is a major challenge in modern cities. Traffic simulation provides a safe environment for testing traffic control strategies before implementing them in the real world.

This project simulates vehicle behavior at intersections and evaluates how different traffic signal strategies impact overall traffic flow.

Traffic simulation systems are widely used in research and urban planning to analyze traffic patterns and optimize road infrastructure.

# 🛠 Tech Stack

* **Python 3**

* **Matplotlib**

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Calebgesami/trafficsimulation.git

```

Create a virtual environment:

```bash
python -m venv .venv
source .venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

** ▶️ Running the Simulation
**
Run the program using:

```bash
python main.py
```

The simulation will:

1. Generate vehicles
2. Simulate traffic movement
3. Apply traffic signal logic
4. Calculate traffic performance metrics

